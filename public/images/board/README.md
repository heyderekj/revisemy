# Board product shots

Drop full-quality exports here (PNG or WebP preferred). The `/board` page picks the first match:

| File | Used when |
|------|-----------|
| `desktop.png` (or `.webp` / `.jpg`) | Viewports ≥ 640px |
| `mobile.png` (or `.webp` / `.jpg`) | Viewports &lt; 640px |

The page renders without images until these files exist — same pattern as `public/images/hero/`.
